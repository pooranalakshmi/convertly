# convertly
Convertly is a simple and user-friendly web application that provides multiple unit conversion tools and a basic calculator in a single platform. The project is designed to help users quickly perform conversions across different measurement categories and carry out simple calculations.

## Features

### 📏 Module 1: Basic Measurements

#### Length Conversion
- Millimeter (mm) ↔ Centimeter (cm)
- Centimeter (cm) ↔ Meter (m)
- Meter (m) ↔ Kilometer (km)
- Inch (in) ↔ Centimeter (cm)
- Foot (ft) ↔ Meter (m)
- Yard (yd) ↔ Meter (m)
- Mile (mi) ↔ Kilometer (km)

#### Weight Conversion
- Milligram (mg) ↔ Gram (g)
- Gram (g) ↔ Kilogram (kg)
- Kilogram (kg) ↔ Ton (t)
- Ounce (oz) ↔ Gram (g)
- Pound (lb) ↔ Kilogram (kg)

#### Area Conversion
- Square Meter (m²) ↔ Square Kilometer (km²)
- Square Meter (m²) ↔ Square Foot (ft²)
- Square Foot (ft²) ↔ Square Inch (in²)
- Acre ↔ Hectare

#### Volume Conversion
- Milliliter (mL) ↔ Liter (L)
- Liter (L) ↔ Cubic Meter (m³)
- Liter (L) ↔ Gallon (gal)
- Liter (L) ↔ Cubic Centimeter (cc)

---

### ⏱️ Module 2: Time & Motion

#### Time Conversion
- Millisecond ↔ Second
- Second ↔ Minute
- Minute ↔ Hour
- Hour ↔ Day
- Day ↔ Week
- Week ↔ Month
- Month ↔ Year

#### Speed Conversion
- Meter/second (m/s) ↔ Kilometer/hour (km/h)
- Kilometer/hour (km/h) ↔ Miles/hour (mph)
- Knot ↔ km/h
- Knot ↔ mph

---

### 🔬 Module 3: Scientific Conversions

#### Temperature Conversion
- Celsius (°C) ↔ Fahrenheit (°F)
- Celsius (°C) ↔ Kelvin (K)
- Fahrenheit (°F) ↔ Kelvin (K)

#### Data Storage Conversion
- Bit ↔ Byte
- Byte ↔ Kilobyte (KB)
- KB ↔ Megabyte (MB)
- MB ↔ Gigabyte (GB)
- GB ↔ Terabyte (TB)

#### Energy Conversion
- Joule (J) ↔ Kilojoule (kJ)
- Joule (J) ↔ Calorie (cal)
- Kilocalorie (kcal) ↔ Joule (J)
- kWh ↔ Joule (J)

#### Power Conversion
- Watt (W) ↔ Kilowatt (kW)
- Kilowatt (kW) ↔ Megawatt (MW)
- Horsepower (hp) ↔ Watt (W)

#### Pressure Conversion
- Pascal (Pa) ↔ Kilopascal (kPa)
- Pascal (Pa) ↔ Bar
- Pascal (Pa) ↔ PSI
- Atmosphere (atm) ↔ Pascal (Pa)

---

### 💰 Module 4: Financial Conversions

#### Currency Converter
- INR ↔ USD
- INR ↔ EUR
- INR ↔ GBP
- USD ↔ EUR
- USD ↔ GBP
- Supports dynamic currency pair conversion

#### EMI Calculator

Inputs:
- Loan Amount
- Interest Rate
- Loan Tenure (Months/Years)

Outputs:
- Monthly EMI
- Total Interest Payable
- Total Amount Payable

---

### 🧮 Basic Calculator

Convertly also includes a simple calculator that supports:
- Addition
- Subtraction
- Multiplication
- Division
- Percentage calculations
